# Megaten Style Dungeon Prototype

女神転生 / Wizardry 方式の 90 度回転 3D ダンジョン RPG のテスト環境です。

## 現在の主な内容

- B1F / B2F / 街の仮マップ
- 入口、階段、宝箱、回復ポイント、ボス、街施設の仮配置
- 宝箱から薬草を入手し、開封済み表示へ変更
- B2F のランダムエンカウントと固定ボス戦
- 3人パーティの HP / MP / 状態表示
- 攻撃、魔法、防御、装備、アイテム、逃げるの仮戦闘コマンド
- 戦闘中の行動ログ、勝利後の探索復帰
- フィールドメニュー、ステータス、アイテム、装備、設定ウィンドウ
- 簡易 BGM / SE と音量設定
- 固定アスペクト比での拡大縮小レイアウト

## 操作

- 矢印キー / WASD: 移動・回転
- Z / Enter / Space: 決定・調べる
- X / Escape: キャンセル
- C: フィールドメニューへフォーカス
- 画面右上ボタン: スマホ向けの回転・前進操作

## 音声差し替え方針

将来的に実ファイルへ差し替える場合は、`sound/field.mp3`, `sound/battle.mp3`, `sound/boss.mp3`, `sound/town.mp3`, `sound/decide.wav` のように `sound` フォルダへ配置し、JavaScript 側の音声設定をファイル再生方式へ切り替える想定です。
